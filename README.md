# dudzik.co

[![Build Status](https://travis-ci.org/doodzik/dudzik.co.svg?branch=master)](https://travis-ci.org/doodzik/dudzik.co)

## Installation

```bash
$ npm install
```

## Usage

to serve the website localy
```
$ make
```

to build the website
```
$ make build
```

to add a new post
```
$ cd dudzik.co
# for a minimalism post
$ ./bin/mini $TITLE
# for a development post
$ ./bin/dev $TITLE
```
