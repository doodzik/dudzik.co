# dudzik.co

[![Build Status](https://travis-ci.org/doodzik/dudzik.co.svg?branch=master)](https://travis-ci.org/doodzik/dudzik.co)

## Installation

```bash
make install
```

## Usage

serve the website localy
```
$ make
```

build the website
```
$ make build
```

add a new blog post
```
$ make blog TITLE="mini"
```

add a new development post
```
$ make dev TITLE="dev"
```
